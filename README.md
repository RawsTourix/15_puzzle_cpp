# 15 puzzle (C++ / SFML)

Классические «Пятнашки» на C++ — учебный проект, чтобы познакомиться с **SFML** и освежить практику **C++**.

## Возможности

- Поле 4×4 (15 тайлов + пустая клетка).
- Управление мышью: клик по тайлу → перемещение, если тайл соседний с пустой клеткой.
- Анимация перемещения (движение по одному тайлу за раз — как в физических пятнашках).
- Перемешивание через последовательность **легальных** ходов (получается решаемое состояние).
- Масштабирование и центрирование поля при изменении размеров окна: поле остаётся квадратным.
- Поддержка «скина» через tilemap/atlas: одна картинка режется на сетку 4×4.
- Обработка ошибок загрузки ресурсов (с выводом сообщения пользователю).

## Требования

- **C++17**
- **SFML 3.0.2**
- Windows (в текущей реализации используются MessageBox для ошибок/уведомлений)

## Сборка

Инструкции по сборке находятся в файле **[BUILD.md](BUILD.md)**.

## Скины (tilemap)

Игра использует одну картинку tilemap, которая режется на 4×4.

Требования к кастомному tilemap:
- соотношение сторон **1:1**
- ширина и высота **делятся на 4** без остатка

> Путь к файлу задаётся в `assets.h`. Чтобы поменять скин — замените файл по этому пути или укажите другой.

## Управление

- **ЛКМ** — попытка сдвинуть выбранный тайл
- **Esc** — пауза

## План развития

- Меню настроек: скорость перемещения тайлов, диапазон перемешивания.
- Сохранение настроек (последние выбранные параметры).
- Рекорды времени (лучший результат / таблица).
- Работа с файлами:
  - сохранение/загрузка текущей партии (позиции, таймер, состояние);
  - выбор кастомных скинов (tilemap) с проверками (1:1 и делится на 4×4) и запоминанием последнего выбранного.
